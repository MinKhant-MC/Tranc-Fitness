# Tranc Gym SSR Version

This folder is the SSR-style version of the Tranc Gym web app.

The original static web app stays in:

`C:\Users\YH\OneDrive\Desktop\MMC MMC\gym-system`

This SSR copy is in:

`C:\Users\YH\OneDrive\Desktop\MMC MMC\gym-system-ssr`

Run it:

```powershell
cd "C:\Users\YH\OneDrive\Desktop\MMC MMC\gym-system-ssr"
npm start
```

Open:

`http://localhost:3000`

Routes:

- `/` or `/login`
- `/register`
- `/member`
- `/admin`

Live data still comes from the Google Apps Script URL in `config.js`.
